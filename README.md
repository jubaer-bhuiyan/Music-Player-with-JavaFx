# JavaFX Music Player

A simple music player application built with JavaFX.

## Features
- Play, pause, and stop music
- Playlist management
- User-friendly interface

## Getting Started
1. Clone the repository
2. Open the project in your preferred Java IDE
3. Build and run the application

## Requirements
- Java 11 or higher
- JavaFX SDK
